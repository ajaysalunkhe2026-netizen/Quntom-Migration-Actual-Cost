# Quntom Migration – Actual Cost

Hybrid Adaptive Migration Framework

## Overview
This project demonstrates a small-scale hybrid migration framework using:

- System Monitoring
- Adaptive Migration Logic
- Optimization Engine
- Logging and Reporting
- Dummy File Migration Testing

The framework is designed as a research-oriented prototype for studying migration behavior, monitoring metrics, and optimization strategies.

## Features
- CPU/RAM/Disk Monitoring
- File Migration Engine
- Optimization Decision System
- Logging Support
- Lightweight Experimental Framework

## Technologies
- Python
- psutil
- Flask

## Project Structure
- monitor.py
- migration_engine.py
- optimizer.py
- logs/
- database/
- dummy_files/

## Current Results
- Monitoring Functional
- Migration Successful
- Optimization Logic Active
- Logging Enabled

## Research Goal
To study adaptive migration behavior and actual operational cost in a lightweight experimental environment.
